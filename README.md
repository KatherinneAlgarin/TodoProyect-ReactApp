# Proyecto TODO realizado para la certificación de Platzi-Curso React.js

## Guía de Ejecución para usar como base
- Clonar el repositorio con `git clone`
- Instalar las dependencias con `npm i` y ejecutar la aplicación `npm start`.

En el directorio del proyecto clonado, puedes ejecutar:

### `npm start`

Ejecutar la aplicacion en modo de desarrollo
Abrir [http://localhost:3000](http://localhost:3000) para verlo en el navegador.

La página se recargará si realizas cambios y lo guardas.
